# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-04-10

### Added
- Initial release
- Google Scholar data entry with pre-built search links
- Journal-specific data entry with source-filtered search links
- Line chart, bar chart, and stacked area charts (Chart.js)
- Configurable search terms, journals, and year range
- Data persistence via browser localStorage
- CSV and JSON export
- JSON import for data restore
- Print-friendly stylesheet
- Responsive design for mobile and desktop
- ARIA attributes and keyboard accessibility
- Methodology guide with links to alternative tools
- Sample data for demonstration
