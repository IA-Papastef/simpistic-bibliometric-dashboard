# Contributing

Thanks for your interest! A few things to keep in mind:

**Keep it as one file.** The whole point is that someone can download `index.html` and it just works. No build tools, no package managers, no splitting into multiple files unless there's a really good reason.

**Test in at least two browsers** before submitting (Chrome + Firefox is enough).

**External libraries** should be loaded via CDN. Include an SRI hash if possible.

**Some ideas for things that would be nice to have:**
- Dark mode
- More chart types (heatmap, maybe?)
- Better mobile experience
- Keyboard navigation improvements
- Translations

To contribute: fork, branch, make your changes, open a PR. Nothing fancy.

If you find a bug, just open an issue with steps to reproduce.
